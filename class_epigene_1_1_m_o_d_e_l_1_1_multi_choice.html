Detailed Description

Thsi class represents a question - answer pairs Questiongs taken from the localization db. Every question has 3 answers, each of these has a prefix in the id which determines the type. Q = question G = Good answer B = Bad answer N = Neutral answer Example: id = 001 Q001="What?" G001="Can I ask something?" B001="Leave me alone!" N001="nothing.."
